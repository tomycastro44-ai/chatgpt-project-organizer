ChatGPT Project Organizer — OpenAI Build Week 2026 Demo

## Description

ChatGPT Project Organizer turns exported AI conversations into structured, evidence-backed and reversible projects.

The demo shows:

- immutable JSON/CSV/TXT import;
- deterministic evidence extraction;
- GPT-5.6 semantic project reconstruction through the OpenAI Responses API;
- global proposals and review by exception;
- mandatory before/after preview;
- simulated application, audit history and exact Undo.

Built with Codex, GPT-5.6, React, TypeScript, FastAPI, Pydantic, SQLAlchemy and SQLite.

Independent prototype created for OpenAI Build Week 2026. It is not an official OpenAI or ChatGPT feature.
