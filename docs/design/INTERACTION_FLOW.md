# Flujo de interacción O.T. 007

```text
UNORGANIZED
    ↓ Organize with AI
IMPORT
    ↓ Validate and import
ANALYSIS
    ↓ Automatic completion
GLOBAL PROPOSAL
    ├── Apply safe proposals
    └── Review evidence
             ↓
PROJECT MEMORY
             ↓
EXCEPTIONS
    ├── Approve
    ├── Correct
    ├── Reject
    └── Keep unclassified
             ↓
PREVIEW
    ↓ Apply approved changes
ORGANIZED RESULT
    ↓
HISTORY & UNDO
    ↓ Confirm undo
RESTORED DERIVED STATE
```

## Regla de navegación

No existe un acceso directo desde importación a aplicación. La propuesta, las excepciones y la vista previa son pasos obligatorios del recorrido demostrable.
