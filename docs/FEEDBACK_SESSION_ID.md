# Codex /feedback Session ID

This identifier must come from the primary Codex build thread.

## Required action

1. Open the Codex thread where most core functionality was built.
2. Run `/feedback`.
3. Copy the generated Session ID.
4. Paste it below and into the Devpost submission form.

`CODEX_FEEDBACK_SESSION_ID=ADD_BEFORE_SUBMISSION`

Do not invent or substitute an unrelated thread identifier.
