# Current limitations

- The MVP imports exported or demonstration conversations; it does not connect to a real ChatGPT account.
- It never modifies actual ChatGPT chats or projects.
- DEMO semantic results are precomputed for reproducibility.
- LIVE semantic reconstruction requires a valid OpenAI API key and was not exercised in the packaged validation environment.
- The application has no authentication, multi-user support or cloud synchronization.
- SQLite is suitable for local evaluation, not a high-concurrency deployment.
- Proposal application affects only a simulated derived workspace.
- The interface is optimized for the competition demonstration rather than full production administration.
