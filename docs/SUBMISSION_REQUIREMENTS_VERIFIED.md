Official sources:

- Rules: https://openai.devpost.com/rules
- FAQ: https://openai.devpost.com/details/faqs
- Schedule: https://openai.devpost.com/details/dates

## Deadline

- Submission closes: **July 21, 2026 at 5:00 PM Pacific Time**.
- Equivalent in mainland Spain: **July 22, 2026 at 02:00 CEST**.

## Required submission items

- Working project built with Codex and GPT-5.6.
- One category/track.
- English text description.
- Public YouTube video under three minutes.
- Video audio explaining what was built and how Codex and GPT-5.6 were used.
- Code repository URL.
- Public repository with a relevant license, or private repository shared with `testing@devpost.com` and `build-week-event@openai.com`.
- README with setup, sample data, testing guidance, Codex contribution, key decisions and GPT-5.6 integration.
- `/feedback` Codex Session ID from the primary build thread.
